### Branches

<p style="font-size:25px;text-align:left;"> At each point you can easily create your new branch using following command:</p>
<pre>
<code>
git branch develop
</code>
</pre>

<p style="font-size:25px;text-align:left;"> Using branches you can for example store working releases of an application in a master branch and development line in *develop*
one </p>
