<pre>
<code>

</code>
</pre>

<p style="font-size:25px;text-align:left;"> 1. Push changes to remote repository</p>
<pre>

</code>
</pre>

<p style="font-size:25px;text-align:left;"> 1. Pull changes from remote repository</p>
<pre>
<code>

</code>
</pre>
